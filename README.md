# Login Page

A simple full-stack **Login Page** application built using **React.js** for the frontend and **Express.js** for the backend. This project demonstrates user login functionality with client-server communication.

## 🚀 Features

* User Login Interface
* React.js Frontend
* Express.js Backend
* API Integration
* Responsive UI
* Easy-to-understand project structure

## 🛠️ Tech Stack

### Frontend

* React.js
* HTML5
* CSS3
* JavaScript (ES6+)

### Backend

* Express.js
* Node.js
